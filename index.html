Java

Java의 객체지향 프로그래밍 원칙을 이해하고 이에 맞는 프로그래밍을한 경험이 있습니다.

Kotlin

Kotlin을 이용하여 어플리케이션을 개발한 경험이 있습니다.

JavaScript

HTML과 CSS와 결합하여 동적인 웹 페이지를 만들 수 있으며, DOM을 조작하여 사용자와 상호작용할 수 있습니다.

기본적인 문법과 동기 비동기 함수에대한 이해가 있습니다.

TypeScript

TypeScript를 사용하여 웹 애플리케이션을 개발한 경험이 있습니다. 특히, React와 TypeScript를 결합하여 컴포넌트 구조를 구현하고 타입 안전성을 강화한 프로젝트를 진행한 경험이 있습니다.

제네릭, 인터페이스를를 적극적으로 활용하여 코드의 재사용성을 높이고, 타입 오류를 컴파일 타임에 잡을 수 있어 개발 속도와 품질을 모두 향상시킨 경험이 있습니다.

SpringBoot

Spring boot를 이용하여 BackEnd구축 경험이 있습니다. Spring Security를 이용한 사용자 인가 인증 구현과 , MySQL MongoDB등을 연동하고 JPA를 이용하여 CRUD 를 구현한 경험이 있습니다.

React

Spring Boot와 함께 연동하여 FrontEnd 구축 경험이 있습니다. 또한 타입스크립트와 함께 컴포넌트의 재사용성을 높히고 여러 hook을 이용한 경험이 있습니다.

주요기능 소개

사용자 인증

  • FrontEnd

    안드로이드 로그인 Activity 구현

    안드로이드 사용자 식별과 인증 토큰 저장

  • BackEnd

    Spring security 와 Filter를 이용하여 사용자 인증 인가 구현

    JWT token을 이용해 인가 절차 구현

    AccessToken과 RefreshToken의 발급 구현

고객관리

  • FrontEnd

    고객추가 ,상세정보, 삭제 Fragment 작성

    NaverMap을 이용한 고객 위치정보

  • BackEnd

    Customer Entity의 관리및 저장

    NaverGeoCoder를 이용한 고객 좌표 정보와 도로명주소 및 지번주소 저장

상품관리

  • FrontEnd

    상품 카테고리 조회,삭제,추가 Fragment 작성

    Retrofit을 이용한 요청 및 ViewModel을 이용한 Data관리

  • BackEnd

    Product Entity의 관리및 저장

    Product , Category Entity의 관리 및 Exception 처리

배달관리

  • FrontEnd

    배달관리 Activity 작성 및 배달정보 리스트 Fragment작성

  • BackEnd

    배달주문 ,배달 완료 및 대기중 관리
